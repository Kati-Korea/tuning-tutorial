01. 서론·학습 목표

운전 인터페이스의 진화와 MFSW의 가치
  • 가치: 안전·조작성·연결성·UX
  • 목표: 하드웨어/전장 이해→장착→코딩→검증
  • 로드맵: 호환성 점검→SOP→코딩/초기화→DTC 0

02. 시스템 아키텍처

  • 하드웨어: 휠·스위치·에어백·클럭스프링
  • 전장: BCM/ICM/Radio/CCU 연동
  • 네트워크: CAN/LIN·게이트웨이

03. 호환성 전략

  • 차종/연식/트림별 스위치 매칭
  • ECU/클럭스프링/커넥터 핀아웃 확인
  • 어댑터 하니스·에뮬레이터 활용

04. 전기/통신 이론

  • 전원/접지·퓨즈·보호소자
  • CAN/LIN 프레임·속도·노드
  • 게이트웨이·신호 에뮬레이션

05. 안전과 인체공학

  • SRS 취급: 배터리 분리·정전기·방향성
  • 토크·정렬·오프센터 방지
  • 버튼 설계: 촉각·야간 시인성

06. 준비와 도구

  • 진단기·스캔툴·보안 접근
  • 핀아웃·배선·단자·압착/리워크
  • 토크렌치·센터링 툴·PPE

07. 장착 SOP

  • 분해: 에어백 탈거·클립 해제·커넥터 분리
  • 교체: 휠 탈거·클럭스프링 정렬·신품 장착
  • 배선: 스위치 하니스·어댑터/에뮬레이터
  • 재조립·규정 토크·초기화·DTC 확인

08. 코딩/적응

  • BCM/ICM/Radio/CCU 파라미터 설정
  • Variant Coding·Security Access
  • 기능 매핑·버튼 테스트·로그 저장

09. 검수/시험

  • DTC 0·SRS 점등 0·셀프 테스트
  • 버튼 매핑 검증·백라이트·햅틱
  • NVH/떨림·오프센터 보정

10. 고급 기능

  • 패들시프트·히티드 휠
  • 햅틱/조명·OTA/암호화 대응
  • 차종별 제약·보증/법규 고려
