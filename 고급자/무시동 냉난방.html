전력 시스템 배선

LFP·DC-DC·MPPT

01. 왜 지금 무시동 냉난방인가

엔진 OFF, 편안함 ON
  • 엔진 공회전 없이 난방/냉방 → 연료 절감, 배출가스·소음 저감, 공회전 제한 조례 리스크 회피
  • 캠핑·차박·상용차 대기·야간 숙영·반려동물/의료 동승 환경에서 피로·위험 감소
  • 배터리 친화 운용과 에너지 예산 기반 운영으로 일관된 쾌적성 확보

02. 시스템 개요: 구성·작동·KPI

  • 히터: 본체(연소실/히트익스체인저/팬), 연료펌프·라인, 흡기/배기, 흡음/방열, 제어패널
  • 에어컨: 컴프레서·콘덴서·증발기·팽창장치, 팬, 제어모듈
  • 전력: 보조배터리(LFP/AGM), BMS, DC-DC 주행충전, MPPT 태양광, 인버터, 차단기/퓨즈/버스바, PMS
  • KPI(히터): 정격 열량[kW], 연료소비[g/h], 전류[A], 승온시간, 소음[dB(A)]
  • KPI(에어컨): 정격 냉량[W 또는 BTU/h], 소비전력[W], COP/EER, 기동전류, 소음, 드레인 성능
  • KPI(전력): 일일 에너지 예산[Wh], 충전 프로파일, SOC·SOH

03. 무시동 히터 심화

  • 연료식(디젤/가솔린): 열밀도 높고 소비전력 낮아 동계 최적. 배기는 외기로 배출, 흡기는 이격 배치.
  • 전기식: 배기가스 없음·설치 단순. 대전력 소모 → LFP 대용량·고출력 DC-DC 필수, 과열/화재 보호 필요.
  • 안전: CO/NOx 차단을 위해 하부 측/후방 하향 배기관, 실내 누기 절대 금지. 연료라인은 규격 피팅·누설시험 필수.

04. 무시동 에어컨 심화

  • 일체형(루프탑): 설치 간편·일체형 배수/밀폐. 루프 컷·보강·방수·하중/무게중심 검토 필요.
  • 분리형: 배치 유연·저중심 설계. 냉매 라인 제작·진공/누설시험·가스 충전 장비 필요.
  • 용량 매칭: 차내 체적·단열·일사량·탑승자/기기 발열 고려하여 냉량 산정.

05. 전력 시스템

  • 배터리: LFP(고사이클·경량·BMS 필요) vs AGM(저비용·저온충전 주의)
  • 용량 산정: 일일 소비[Wh]=Σ(부하W×시간). 권장 용량[LFP Ah]=Wh/12/0.8
  • 충전: DC-DC(차량 보호·분리 충전), 태양광 MPPT, 발전량 시뮬.
  • 분전/보호: 메인퓨즈·개별 보호, 전압강하 <3%, 스타형 접지.
  • 인버터: 정현파, 피크/지속용량과 기동전류 대응.
  • PMS: SOC 모니터, 저전압 차단, 차단기·릴레이 인터록.

06. 열·소음·진동·방수

  • 열: 배기관 이격거리 확보, 방열판/슬리브, 가연물 근접 금지.
  • 소음/진동: 펌프 고무마운트, 플렉서블 배기관, 흡음재·덕트 설계.
  • 방수/방진: 루프 관통부 실링, 드레인 S-트랩, IP 등급 커넥터.

07. 준비·설계 단계

  • 부하/열량 계산, 설비 배치 도면, 관통부 강화(프레임/리브 회피)
  • 부품 스펙시트: 호스/피팅 규격, 케이블 게이지, 퓨즈 선택표, 방수등급
  • 위험분석(HAZOP): 화재, CO, 누설, 쇼트, 침수 시 대책

08. 시공 SOP(히터)

  1. 작업 안전: 연료 증기 환기, 소화기, 배터리 분리
  2. 위치 선정·템플릿 마킹, 하부 관통 드릴링
  3. 본체 고정·가스켓·실링, 흡기/배기 배치(하향·차폭내)
  4. 연료 라인: 탱크 피딩·필터·펌프 방향성·동고·클램프
  5. 전원: 메인퓨즈→릴레이→히터, 제어패널 배선
  6. 시동·초기화: 프라이밍, 연소 안정화, 누기·CO 측정
  7. 문서화: 토크·배관·전기 사진, 체크리스트

09. 시공 SOP(에어컨)

  1. 레이아웃·하중 검토
  2. 루프 컷(일체형): 마스킹→컷→프라이머→보강→실링
  3. 분리형: 실내/실외기 브라켓, 진동 아이솔레이터
  4. 배관/배선: 냉매 라인 플레어/브레이징, 진공/누설시험, 드레인 경사
  5. 전기: 전원 케이블 굵기, 메인퓨즈·차단기, 인버터/직결
  6. 시운전: 과열/결빙, 소음/진동, 누수/결로 점검
  7. 문서화: 냉매량·진공/누설 기록, 소음 측정

10. 시운전·검수·인수

  • 성능: 히터 승온 속도/전류, 에어컨 냉각 속도/소비전력, COP/EER
  • 안전: 실내 CO 0 ppm, 배기 누기 0, 누유/누수 0, 케이블 온도 상승 허용치 이내
  • 소음: 실내/외 dB, 펌프·팬 소음 관리
  • 문서: 회로도, 토크표, 사진 10컷+, 시운전 로그, 인수서

11. 유지관리·정비

  • 히터: 연료필터·버너 카본 청소, 흡기 필터, 배기관 부식 점검
  • 에어컨: 필터 청소, 콘덴서/증발기 세척, 냉매 압력/누설 점검
  • 전력: 배터리 SOC·내저항, 단자 토크·부식, 펌웨어 점검
  • 보증/주기: 사용시간 로깅, 소모품 스케줄 준수

12. 연계 튜닝

  • 태양광: 루프패널·랙 통합, MPPT 매칭, 케이블 루팅, 발전량 예측
  • 주행충전: 고출력 DC-DC(발전기 보호), 열관리
  • 단열/차광: 열부하 저감 → 필요한 냉/난방 용량·소비 전력 감소
  • 소음 저감: 펌프/배기 흡음, 방진 마운트

13. 법령·검사·보험/인증(한국)

  • 튜닝 분류: 무시동 히터/에어컨은 통상 ‘경미한 튜닝’. 단, 루프 컷·구조보강·배기관 외부 배출 구조는 안전기준 적합 필요.
  • 전기·KC: 12/24V 장치 KC 적합성, 전선/퓨즈 규격 준수
  • 소음/주차: 야간 소음 민원 예방, 캠핑장 규정 준수
  • 검사/서류: 시공 사진, 전기/배기 도면, 부품 사양서, 인증서, 점검표
  • 보험/책임: 화재·중독 사고 대비, 사전 고지·작업 증빙·정기 점검 기록
참고: “자동차 튜닝에 관한 규정(국토교통부)”, 지자체 공회전 제한 조례, KC/전기안전 관련 법령 확인

14. NCS 기반 직무역량·평가

  • 요구분석·설계: 부하·열량·전력예산, 배치/관통, 위험분석
  • 전력시스템 구축: 배터리/BMS, DC-DC/MPPT/인버터, 분전/보호
  • 히터/에어컨 시공: 연료·배기·흡기/냉매·드레인, 장착·실링
  • 시운전·검수·문서화: 성능/안전 시험, 인수서 패키지
  • 법규·고객안내: 기준 준수, 보험/책임, 유지관리 교육
  • 전선 굵기/퓨즈 100% 적합, 전압강하 <3%
  • 실내 CO 0 ppm, 누유/누수 0, 절연저항 합격
  • 히터 승온/에어컨 냉각 목표치 ±10% 이내
  • 문서화 100%(회로·토크·사진·로그·서명)

15. 실습 시나리오(예시)

  • 실습 A: 2kW 연료식 히터 장착(4시간) → 관통·배기/흡기·연료 라우팅·CO 0
  • 실습 B: 12V 루프탑 에어컨 설치(6시간) → 루프컷/보강/실링, 드레인 설계
  • 실습 C: 분리형 AC 진공·누설시험(3시간) → -1 bar 30분 유지, 누설 0
  • 실습 D: 전력 패키지(200Ah LFP+40A DC-DC+300W 태양광)(6시간) → 배선/보호/설정
  • 실습 E: 트러블슈팅 챌린지(2시간) → 시동불량·결로 누수·전압강하 진단

16. 트러블슈팅 사례

  • 히터 꺼짐/미점화: 연료 공기혼입·펌프 극성·배기 역풍 → 라인 재정렬·역류방지·프라이밍
  • 연소 냄새/CO: 실링 불량·흡기 재순환 → 실링 재시공·흡기 위치 변경
  • 펌프 딸깍 소음: 고정 불량·공명 → 아이솔레이터·흡음재
  • 에어컨 냉방 약함: 일사/단열 부족·냉매 부족 → 차광·단열·충전
  • 결로/누수: 드레인 경사/막힘 → 경사 재설계·청소
  • 전력 부족: 인버터 피크·케이블 과소·전압강하 → 굵기 상향·배선 재시공

17. 고객 커뮤니케이션·문서

  • 안전: CO 경보기 점검, 환기 원칙, 연료/배기 주의, 야간 소음 매너
  • 운용: 히터 예열/후열, 필터 청소, 에너지 예산 계산
  • 보증: 소모품·설치부 보증 조건, 정기점검 주기
  • 문서 패키지: 회로/배치도, 부품 스펙, 시운전 로그, 유지관리 카드, 법적 안내문, 사진 12컷+
