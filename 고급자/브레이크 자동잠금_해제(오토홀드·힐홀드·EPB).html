법규 확인

01. 서론: 편의 그 이상 – 안전·제어·피로 저감

일상·경사·정체·트랙에서의 가치
  • 정차 중 홀드, 출발 시 자동 해제로 운전자 피로 저감·롤백/롤어웨이 방지
  • 오토홀드+EPB 인터락으로 시동 OFF·도어 오픈 등 비정상 이벤트에도 안전 확보
  • 도심 신호 대기·경사 주차·피트인/아웃 등에서 일관된 제어 제공

02. 시스템 정의·분류

  • 오토홀드: 유압 라인압 유지 기반 정지 상태 유지, 가속 요청 시 자동 해제
  • 힐홀드: 경사 출발 시 2–3초 라인압 유지
  • EPB 인터락: 조건에 따라 전자식 주차브레이크 자동 체결/해제
  • 롤어웨이 방지: 도어/벨트/기어 변경 등 이벤트에서 자동 홀드

03. 아키텍처 개요

  • 센서: 압력·페달·휠속·IMU(경사)·PRND·도어·벨트
  • ECU: 상태기계, 타임아웃, 다중 센서 융합
  • 액추에이터: 유압 모듈, EPB 모터, 부스터 연계
  • HMI: 경고등·클러스터 메시지·부저
  • 통신: CAN/LIN 네트워크

04. 핵심 원리

  • 정지 판단: 휠속 0·라인압·기어·경사·버튼 입력 결합
  • 경사 추정: IMU·로지컬 필터로 롤백 토크 예측
  • 해제 논리: 가속 요청·클러치 결합·토크 빌드업 시점 연동
  • 페일세이프: 타임아웃·전압저하·센서 불일치 시 안전 정지/알림

05. 구성요소 심화

  • 센서: 압력·페달 스트로크·휠속·IMU·PRND·도어·벨트 신호 품질
  • ECU/알고리즘: 상태기계, 히스테리시스, 타임아웃 테이블
  • 액추에이터: 유압모듈/EPB 모터, 부스터와의 인터랙션
  • HMI: 명확한 표시/부저·가이드 메시지

06. 통합 시나리오

  • ABS/ESC/EPB와 CAN/LIN 통합, 개입 우선순위 관리
  • AT/MT/HEV/EV 별 해제 조건·토크 제어 차이
  • 아이들 스톱·ACC(정차/재출발)와의 상호작용

07. 성능 지표·시험 방법

  • 홀드 신뢰도[%], 해제 반응시간[ms], 롤백 허용치[cm], 편제동[%]
  • 환경 시험: 경사/우천/저온·고온/전압변동

08. 설계 고려: 안전목표·ASIL

  • 안전목표 정의, HARA, ASIL 할당
  • 페일-실런트/페일-세이프 전략, 진단 커버리지

09. 하드웨어 설계

  • 배선·전원·접지·퓨즈, EMI/EMC 대응
  • 방열·방진·커넥터 방수·하니스 보호

10. 소프트웨어/캘리브레이션

  • 임계값·히스테리시스·타임아웃·사용자 설정
  • 차량·타이어·구동계 변화에 따른 맵 보정

11. 설치 SOP

  1. 준비: 배터리 분리·도면/배선 계획·퓨즈 선정
  2. 배선/장착: 하니스 루팅·방열/차폐·고정
  3. 보호: 퓨즈/릴레이·커넥터 실링·아이솔레이션
  4. 상태 학습: 초기화/자세 학습/제어 상태 확인
  5. 검수·시운전: 경사 홀드·해제 반응·경고/메시지

12. 검증 절차

  • 정지/출발/경사/우천/저온·고온·전압변동 케이스
  • 로그: 홀드 신뢰도·반응시간·편제동·경고 이벤트

13. 유지보수/내구

  • 정기 점검·데이터 로거 점검
  • 배터리 상태·EPB 마모·유압 라인 누설 확인

14. 고성능 주행 연계

  • 드리프트/서킷: 개입 레벨/자동해제 로직 관리

15. NVH/인체공학

  • 진동·소음·페달 느낌·HMI 명료성 개선

16. 트러블슈팅

  • 오작동/오해제/오홀드 사례 원인-대응
  • 경고등 점등 진단(센서 불일치·전압저하·통신 오류)

17. 비용/패키지/TCO

  • 등급별 구성·공임·문서화 비용
  • TCO: 유지보수·소프트 업데이트·검증 비용

18. 법령/검사/보험(대한민국)

  • 구조변경·안전기준·전자파(KC)·서류화
  • ADAS/EPB/ESC 연계 시 법적 적합성 확인
  • 보험 고지·책임 범위 명확화
참고: TS 한국교통안전공단 구조변경 가이드와 KC/전자파 적합성, 제동장치 관련 고시를 사전 확인하세요.

19. NCS 기반 역량·평가체계

  • 요구분석·설계: 안전목표/ASIL·아키텍처
  • 시공/통합: 배선·방열·진단·통신
  • 검증/문서화: 시험 케이스·로그·인수서
  • 홀드/해제 KPI 달성, 경고/오류 0, 문서 패키지 100%

20. 부록 A ~ D

  • 부록 A 체크리스트/로그/동의서 템플릿
  • 부록 B 부품·공구·토크 표
  • 부록 C 이미지 키워드·참고 링크
  • 부록 D 용어집·FAQ
