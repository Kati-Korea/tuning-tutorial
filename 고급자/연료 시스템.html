01. 서론: 파워·내구·안전을 결정하는 연료 시스템

출력 한계는 ‘연료 전달’이 결정
  • 안정적 유량·압력 공급은 출력·내구·배출 모두에 직결
  • 튜닝 전 요구유량 산정과 호환성(연료/재료/전기) 검토 필수

02. 연료와 연소의 기초

  • 스토이키오메트리: 가솔린 \[\lambda=1 → AFR≈14.7], E85/디젤 별도
  • 옥탄/세탄, 증기압, 잠열: 노킹/착화/핫스타트에 영향

03. 시스템 아키텍처

  • 탱크 → 프리필터 → 펌프(인탱크/인라인) → 필터 → 레귤레이터 → 레일 → 인젝터
  • 리턴/리턴리스, EVAP 증발가스 제어 구성

04. 요구유량·압력 설계

  • 목표출력 기반 펌프/인젝터 용량: \[\text{Flow} = \text{HP} \times \text{BSFC} / \text{연료밀도}]
  • BSFC 가솔린(NA)≈0.45–0.55, 터보≈0.55–0.65 kg/hp·h, 여유율 20–30%

05. 펌프·필터·레귤레이터

  • 펌프: 인탱크 저소음/냉각 유리, 인라인 증설·서지탱크 연동 용이
  • 필터: 전단/후단 미크론 등급, 차압 모니터링
  • 레귤레이터: 진공 참조, PWM 제어 펌프와의 정합

06. 연료 레일·라인·피팅

  • 호스/피팅 내연료·내열 등급, 유량 대비 압력 강하 최소화
  • 캐비테이션 방지: 베ンド·고온 구간 단축, 방열/차열

07. 인젝터·분사전략

  • 포트/직분, 듀얼 인젝션 구성, 분사 타이밍·패턴 최적화
  • 인젝터 용량/임피던스/데드타임 데이터로 ECU 보정

08. 디젤 시스템 특론

  • 커먼레일 고압펌프/인젝터 코딩, 파일럿/메인/애프터 분사
  • 리턴 제어·연료 온도 관리, 고압 누설 안전

09. 열관리·베이퍼락·냉각

  • 라우팅 최적화, 차열 시트·실드, 리턴 유무에 따른 온도 전략
  • 베이퍼락 예방: 탱크 환기/압력 관리, 열원 차단

10. ECU·센서·제어

  • MAP/MAF/람다/FRP 센서 활용, 클로즈드 루프 연료 제어
  • 펌프 PWM, 연료압 타겟 맵, 보호 로직(저압/저유량)

11. 진단·트러블슈팅

  • 연료압 흔들림, 딜리버리 부족, 미스파이어, 핫스타트 대응
  • 게이지/스코프/로그로 원인 분리(전기·유압·열)

12. 모터스포츠·고성능 설계

  • 서지탱크, 듀얼 펌프, 체크밸브, 스위치오버 제어
  • E85/메탄올 호환: 용량 보정, 재료 호환성, 콜드 스타트 전략

13. 안전·방폭·규정

  • 연료 누출/화재 예방: 호스 규격·클램프 토크·퓨즈/릴레이
  • 정전기·점화원 관리, 작업·보관 안전 수칙

14. 법령·검사·보험(대한민국)

  • 구조변경·배출·증발가스(EVAP) 관련 적합성 검토
  • 변경사항 서류화·보험 고지·안전 라벨
참고: TS 한국교통안전공단 구조변경 가이드 및 연료계통 관련 고시, 증발가스 규정(K-EVAP)을 사전 확인하세요.

15. NCS 기반 역량·평가

  • 요구유량 산정·부품 선정·배선/배관·캘리브·검증·문서화
  • KPI: 연료압 안정·유량 목표 달성, 누유 0, 문서 패키지 100%

16. 부록 A ~ D

  • 부록 A 체크리스트/서식(설계·설치·PDI)
  • 부록 B 규격·토크·재료 호환 표
  • 부록 C 이미지/레퍼런스 링크
  • 부록 D 용어집·FAQ
